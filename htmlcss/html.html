HTML

//HTML is the standard markup language for Web pages. 

- It supports audio and video tags.
- Allows JavaScript to run in background. 
This is possible due to JS Web worker API in HTML5.
- SVG and Canvas for graphics
- HTML5 is mobile friendly
- Supported by all browsers
- News elements for web structure like nav, headers footer
