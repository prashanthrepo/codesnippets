Hooks

React Hooks are in-built methods that allow developers to 
use state and lifecycle methods inside functional components
(basic and stateless functions). - Improved code reuse.
Why Hooks
Reusing stateful logic between components is difficult.
When components become larger and carry out many operations, 
it becomes difficult to understand in the long run.
Hooks Rules
Hooks should not be used inside loops, conditions, or nested functions;
Hooks should be used inside React Functions.



useState Hook - reactive values (any datatype),

// State for the component
import React, { useState } from "react";
function app() {
  const [name, setName] = useState("Prashanth");
  handleClick = () => {
    setName("Srikanth");
  };
  return (
    <div>
      <p>{name}</p>
      <button onClick={handleClick}>Click</button>
    </div>
  );
}
export default app;



useEffect Hook- runs on every render.

//useEffect runs on every render.
//Just like component did mount.

import {useEffect} from 'react'
let name = 'hello'
useEffect(() => {
    console.log('Hello')
}, []);

//Second empty parameter to run the useeffect only once.
[name] = > Runs on change of variable.
useEffect(() => {
    console.log('Hello')
}, [name]);



useRef Hook

//useRef he useRef Hook allows us to create 
//mutable variables in functional components.

function TextInputWithFocusButton() {
  const inputEl = useRef(null);
  const onButtonClick = () => {
    // `current` points to the mounted text input element
    inputEl.current.focus();
  };
  return (
    <>
      <input ref={inputEl} type="text" />
      <button onClick={onButtonClick}>Focus the input</button>
    </>
  );
}



useParam Hook - to access attributes of an url

import React, { useParams } from 'react-router-dom'
const Blogdetails = () => {
  const { id } = useParams()
  return (
    <div> className="blog-details">
      <h2> Blog details : {id} </h2>
    </div>
  )
}
export default Blogdetails;


useHistory Hook - to access history

import React, { useHistory } from 'react-router-dom'
const goback = () => {
  const history = useHistory()
  history.go(-1)
  history.push('/home');
}
export default goback;
