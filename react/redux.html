Redux - Central data store

//Install - npm i redux react-redux
//Redux is central data store for all of the application data
//Any component can access data from this store.
//Makes state management very easy
//Reducer - is a function which interacts with the store


Import Redux in index.js (root)

import { createStore } from 'redux'
import rootReducer from './path'
import { Provider } from 'react-redux';
//Provider - bind redux with react
const store = createStore(rootReducer);

ReactDOM.render(<Provider store={store}><App/>
</Provider>, document,getElementById('root'))


Reducer - is a function which interacts with the store

Combine Reducers - combine two reducers into one

import authReducer from './path'
import projectReducer from './path'
import { combineReducers } from 'redux'

const rootReducer = combineReducers({
  auth : authReducer,
  project : projectReducer
})

export default rootReducer;
